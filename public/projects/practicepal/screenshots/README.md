# PracticePal Screenshots

Place your PracticePal UI screenshots in this folder. The landing page is configured to reference these files in order:

## Expected Files

- **01-hero.png** - Hero/onboarding screen (Slot 1)
- **02-dashboard.png** - Dashboard overview (Slot 2)
- **03-session-logging.png** - Session logging flow (Slot 3)
- **04-billing.png** - Billing and upgrade flow (Slot 4)

## Image Requirements

- Format: PNG, JPG, or WebP
- Aspect ratio: 4:3 (e.g., 800x600, 1200x900)
- Recommended size: 1200x900 (will be responsive)
- Optimize for web to reduce file size

## Example

```
screenshots/
├── 01-hero.png
├── 02-dashboard.png
├── 03-session-logging.png
├── 04-billing.png
└── README.md (this file)
```

These will appear on the archived demo page at `/projects/practicepal/landing`
